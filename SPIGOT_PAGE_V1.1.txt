[SIZE=5][B][COLOR=#3366FF]üì¶ What is DeliveryCore?[/COLOR][/B][/SIZE]

DeliveryCore is a powerful and flexible delivery event plugin that allows server owners to create scheduled item collection events. Players compete to deliver specific items within a time limit, with automatic rewards for top performers.

[SIZE=5][B][COLOR=#3366FF]‚ú® Features[/COLOR][/B][/SIZE]

[LIST]
[*][B]Scheduled Events[/B] - Daily, weekly, monthly scheduling with cron-like syntax
[*][B]Random/Fixed Items[/B] - Random item selection from categories or fixed items
[*][B]Multiple Categories[/B] - Ore, Farm, Block, and custom categories
[*][B]Automatic Rewards[/B] - Items, commands, or both for winners
[*][B]Discord Webhooks[/B] - Start, end, and warning notifications
[*][B]Beautiful GUI[/B] - Clean inventory-based interface
[*][B]Multi-language[/B] - Turkish and English support
[*][B]Tab List Integration[/B] - Show active events in player tab (TAB plugin compatible)
[*][B]Season System[/B] - Restrict events to specific months
[*][B]Custom Items[/B] - Add custom items with NBT data
[*][B]Chest Delivery[/B] - Deliver from chests with protection support
[/LIST]

[SIZE=5][B][COLOR=#3366FF]üìã Commands[/COLOR][/B][/SIZE]

[CODE]
/dc help              - Show help menu
/dc reload            - Reload configuration
/dc list              - List all deliveries
/dc status [name]     - Show active event status
/dc start <name> [duration] [winners] - Start event manually
/dc stop <name>       - Stop active event
/dc top [name]        - Show leaderboard
/dc tab <name> on/off - Toggle tab display
/dc additem <cat> <name> [price] - Add held item to category
/dc removeitem <cat> <name> - Remove custom item
/dc listcustom        - List custom items

/teslimat             - Open delivery GUI (player)
/teslim               - Deliver items from inventory
/teslim sandik        - Deliver from chest
[/CODE]

[SPOILER="deliveries.yml example"]
[CODE=YAML]
deliveries:
  gunluk:
    enabled: true
    display-name: "Daily Delivery"
    winner-count: 3
    
    category:
      mode: random
    item:
      mode: random
    
    timezone: Europe/Istanbul
    
    schedule:
      start: "every day 20:00"
      end: "every day 21:00"
    
    rewards:
      1:
        type: both
        item: DIAMOND
        amount: 10
        commands:
          - "eco give {player} 5000"
      2:
        type: item
        item: DIAMOND
        amount: 5
      default:
        type: command
        commands:
          - "eco give {player} 100"
    
    webhook:
      enabled: true
    
    tab-display:
      enabled: true
[/CODE]
[/SPOILER]
